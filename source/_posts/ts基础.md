ts运行环境
全局安装 ``npm install typescript -g``
编译``ts tsc 01.ts``

简化 类似预览ts
``npm install ts-node -g``
之后用 ts-node 01.ts  就可以不编译预览

创建项目 
``1.npm init -y //创建package.json``
``2.tsc --init``
``3.npm install typescript -D  //当前目录安装ts``
